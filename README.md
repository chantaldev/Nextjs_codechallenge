# Multi-step form



## Tecnologías Utilizadas

- **Lenguaje**: TypeScript
- **Framework**: Next.js
- **Validación**: Zod
- **Estilos**: Styled-components
- **Gestión de Estado**: Zustand
- **API Simulator**: Beeceptor
